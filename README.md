### Running Locally

Requires [Node.js](https://nodejs.org/en/) <br />
`$ git clone https://github.com/BlakeIsMyWaifu/PaydayBuilder` <br />
`$ yarn install` <br />

To build and run a production build <br />
`$ yarn build` <br />
`$ yarn start` <br />
`$ yarn open-browser` <br />

To start a development server <br />
`yarn dev`

#### Technologies

- Node.js
- TypeScript
- React
